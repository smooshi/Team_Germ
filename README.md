<h1>Team_Germ</h1>

Tehdään heroes 3 might and magic tapainen vuoropohjainen RPG seikkailu/taistelupeli PC:lle, jossa on 3D grafiikat ja kuvattu 3. persoonasta. Käytetään Unity 3D:tä ja tehdään C# kielellä.

Teemana on bakteeri/virus rpg/seikkailu
